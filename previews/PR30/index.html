MixedModelsDatasets._download — Method
_download(nm::AbstractString; info=true)

Return the local cache path for dataset nm, downloading it (and verifying its SHA-256 checksum) if it is not already cached or the cached file's checksum no longer matches. Set info=false to suppress the @info log messages.
